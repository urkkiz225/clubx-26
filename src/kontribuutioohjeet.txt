windowsille ohjeet siis
lataa node.js, sen mukana tulee komentorivityökalu npm = node package manager

JOS koneella on github käyttikset terminaalissa (ei version.aalto.fi käyttikset:)
kloona repository joka on toivottavasti ajan tasalla
navigoi komentorivillä kloonattuun kansioon
runaa npm install nii saat kaikki dependencyt pystyyn (saatta mennä pari minaa)
runaa npm run start se pistää webpackin pystyyn
luo uusi local branch koneella komennolla git checkout -b (joku-hieno-uuden-branchin-nimi)
tee muutokset
git add
git commit

sit
git push -u origin (joku-hieno-uuden-branchin-nimi)

voit tän jälkeen joko ite avata githubis uuden pull requestin 
eli käytännössä vaa pyydät mergaamaan sun tekemän koodin tahi muun main branchiin

tai sit jos oot valmis kaiken koodin kaa voin ite mergaa sen master branchiin samalla tapaa

JOS koneella on ainoostaan version.aalto.fi käyttikset terminaalissa:
tällä metodilla on vähän hankalampaa lisätä tosi monta filee mut ei täs kovin montaa filee muokatakkaan paitsi jos tyylii installaa jonkun ulkosen libraryn
kloonaa repositorio lataamalla se repositorion nettisivult
navigoi folderiin cmds
runaa npm install nii saat kaikki dependencyt pystyyn (saatta mennä pari minaa)
runaa npm run start se pistää webpackin pystyyn
tee sun muutokset koodissa
mee repositorion nettisivulle
tee kohast branches -> view all branches -> new branch uus branchi (pitäs pystyy ku on contributer rooli)
lataa sun muokatut tiedostot sinne

tee samalla tavalla pull request tai sit mä voin mergaa koodin masteriin ihan kummin vaa

voi myös lisää main branchiin uusii tiedostoi mut yritä olla pistämät koodimuutoksii suoraan main branchiin pls xd
